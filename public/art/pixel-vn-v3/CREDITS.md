# Pixel VN V3 Assets

All images in this directory were generated as original project artwork for
the medical-English burger game on 2026-05-27 and locally reduced to a limited
palette WebP pixel presentation.

- `characters/*`: original medical-student portrait states, manually selected
  for distinct props, poses and visible two-arm anatomy.
- `characters/bun/normal1.png`: user-supplied regenerated normal frame for the
  radiology student.
- `characters/bun/normal1.webp`: locally resized high-quality WebP derived from
  `normal1.png` for in-game normal/waiting display.
- `boss/*`: original full-width female professor Boss finale storyboard
  frames. These are horizontal finale/theater frames only, not desktop battle
  portraits.
- `boss-portrait/*`: reserved for future vertical female professor Boss battle
  portraits. Do not substitute old male-professor assets, do not use
  `pixel-characters/professor-boss.png`, and do not expand/crop horizontal
  finale frames into this role.
- `scenes/shop-background.webp`: original brighter night-shift diner backdrop.

No third-party game character artwork, commercial music or external textures
are included in this asset set.
