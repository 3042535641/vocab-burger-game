# Pixel VN V3 Assets

All images in this directory were generated as original project artwork for
the medical-English burger game on 2026-05-27 and locally reduced to a limited
palette WebP pixel presentation.

- `characters/*`: original medical-student portrait states, manually selected
  for distinct props, poses and visible two-arm anatomy.
- `boss/*`: original full-width Professor Qiu finale storyboard frames.
- `scenes/shop-background.webp`: original brighter night-shift diner backdrop.

No third-party game character artwork, commercial music or external textures
are included in this asset set.
